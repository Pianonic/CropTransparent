About CropTransparent

The fast and simple transparent image cropping tool

What is CropTransparent?

CropTransparent is a lightweight web application designed to automatically crop transparent areas from PNG, GIF, and WEBP images. It's built with a focus on speed, simplicity, and privacy.

How It Works

The application uses the Python Imaging Library (PIL/Pillow) and NumPy to:

  1. Convert images to RGBA format if needed
  2. Extract the alpha channel
  3. Find the bounding box of non-transparent pixels
  4. Crop the image to this bounding box
  5. Deliver the result directly to your browser

Privacy & Security

Your images never leave your computer. All processing happens in memory on the server, and no files are stored permanently. The application uses direct memory-to-memory transfer to ensure your images remain private.
